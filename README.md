# hrbs-backend

- Seeding dataset - [dataset](https://www.kaggle.com/datasets/raj713335/tbo-hotels-dataset/data)

Prerequisites:

- [Vercel CLI](https://vercel.com/docs/cli) installed globally

To develop locally:

```
npm install
vc dev
```

```
open http://localhost:3000
```

To build locally:

```
npm install
vc build
```

To deploy:

```
npm install
vc deploy
```

To spin up pg db:

```
docker compose up -d
```

Convert .env.example to .env

```
mv .env.example .env
```
